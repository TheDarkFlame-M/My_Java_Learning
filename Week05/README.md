## 学习路径

### 初学者路径
1. **先学分支**：从`IfBasicDemo`开始 → `BranchControlDemo` → `WeekdaySwitch`
2. **再学循环**：`LoopBasicDemo` → `LoopDetail` → `BreakContinueDemo`
3. **基础练习**：完成`IfElseExercise`和基础循环练习

### 进阶者路径  
1. **算法思维**：`LoopAdvanced`中的素数判断、斐波那契数列
2. **模式识别**：`MultiplicationTable`和`Pyramid`的图案生成
3. **项目实战**：`GradeStatistics`和`LoginSystem`完整系统
