# Java 数组操作实践

系统整理 Java 数组相关的核心概念与典型操作，涵盖一维/二维数组的声明、初始化、内存模型、动态管理及常用算法。所有示例可独立运行，适用于基础巩固或教学参考。

---

## 目录结构
	java-arrays/
	├── ArrayBasic.java              // 基础声明、初始化、遍历
	├── ArrayBasics.java             // 声明陷阱、类型约束、编译错误示例
	├── ArrayAssignmentDemo.java     // 值拷贝 vs 引用传递对比
	├── ArrayAlgorithms.java         // 查找、拷贝、反转、扩容等工具方法
	├── ArraySearchInsert.java       // 顺序查找 + 有序插入（保持升序）
	├── InteractiveArrayManager.java // 用户交互式动态数组管理
	└── Array2DExamples.java         // 二维数组：规则/锯齿结构、杨辉三角

## 核心内容

### 基础机制
- 静态与动态初始化
- 数组默认值行为（`int` → 0, `boolean` → false, 引用 → null）
- 类型一致性约束与常见编译错误（如 `char[]` 误用于 `String[]`）

### 内存语义
- 基本类型赋值：值拷贝，互不影响  
- 数组赋值：引用传递，共享同一对象  
- 验证方式：`arr1 == arr2` 判断是否指向同一实例

### 动态操作
- **扩容**：创建新数组（长度+1），复制原数据，追加元素  
- **缩减**：创建新数组（长度-1），复制除末尾外的所有元素  
- **有序插入**：在升序数组中定位插入点，保持顺序不变

### 算法实现
- 最大值查找（记录值与下标）
- 顺序查找（支持字符串 `.equals()` 比较）
- 数组反转（逆序复制）
- 二维数组求和（支持不规则结构）

### 交互设计
- 用户控制增/删流程
- 实时显示数组状态（内容 + 长度）
- 边界保护（如禁止空数组缩减）
